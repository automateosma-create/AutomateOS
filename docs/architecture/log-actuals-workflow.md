# Log Actuals Workflow

The canonical compact representation of `AutomateOS - Log Actuals` is now the [NWS-1 workflow pack](../workflows/log-actuals/spec.md).

Use the pack for:

- architecture and terminal branches;
- request and response contracts;
- node-map reconstruction status;
- data mutations, matching, idempotency, and audit behavior;
- validation and health requirements;
- explicit missing-artifact status.

The exact active n8n export and Code-node source are not yet preserved in GitHub. The workflow pack labels these gaps rather than reconstructing implementation code from prose. Git history preserves the earlier narrative version.
